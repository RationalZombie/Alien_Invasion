# 👾 Alien Invasion

A retro arcade-style 2D shooter game built from scratch using Python and Pygame. Take control of a starship, defend the bottom screen from an advancing armada of alien fleets.

#### CONTROLS

The game provides intuitive keyboard mapping:

| Key Binding | Game Action |
| :--- | :--- |
| **`←` / `Left Arrow`** | Move ship smoothly to the left |
| **`→` / `Right Arrow`** | Move ship smoothly to the right |
| **`Spacebar`** | Fire laser bullets (Up to 3 concurrent projectiles) |
| **`Enter`** | Instantly launch the game from the title card |
| **`Q`** | Force-quit the application safely at any point |

---

#### HOW TO PLAY


Head over to the `Releases` page on this repository and download the build tailored to your operating system:
* **Windows Users**: Download `alien_invasion.exe` and double-click to launch.
  > *Windows Note*: If your browser has blocked the download, find the `Save` button to actually download the .exe file.
* **macOS Users**: Download `alien_invasion_mac.zip`, unpack it, and open the bundle app.
  > *Mac Note*: If Gatekeeper blocks execution with an unverified developer warning, hold `Control` while clicking the app icon, select **Open**, and confirm "Open Anyway". Alternatively, go to `System Settings` (macOS Ventura and above), head to `Privacy & Security`, scroll to the bottom to allow Alien Invasion to run on this Mac.
